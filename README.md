## About

I build local-first tools for developers, researchers, and long-form creators.
My favorite software is fast, inspectable, and sharp enough to keep up with thought.

- AI coding agents around DeepSeek, Claude, Kimi, and terminal workflows.
- Native-speed knowledge systems with C++, Rust, Tauri, React, and SQLite.
- Practical AI tools for research reading, trading workflows, and long-form writing.
- A strong bias toward performance, clean architecture, and tools that stay useful after the demo.
